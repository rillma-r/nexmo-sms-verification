### This is an implementation of Certification Project using  SMS & Verification API. ###

This repo contains the code for https://vonage-workshop.nexmodev.com/verify/certification/certification-project/ on using the Nexmo Verify and SMS API.

### GitHub Repository ###
You can download your own version from:
https://github.com/rillma-r/nexmo-sms-verification.git

### How download and run the project ###
You can use below commands:
$$\...> git clone https://github.com/rillma-r/nexmo-sms-verification.git
$$\...> cd nexmo-sms-verification 
$$\...\nexmo-sms-verification> npm install

### Initial Configuration ###
Edit the '.env' with your data from https:\\dashboard.nexmo.com

NEXMO_API_KEY=YOUR NEXMO API KEY
NEXMO_API_SECRET=YOUR NEXMO API SECRET
NEXMO_BRAND_NAME=ANY TEXT
VIRTUAL_NUMBER=YOUR VIRTUAL MOBILE NUMBER
 
### Start the application ###
In order to star the application run below command:

$$\...\nexmo-sms-verification> npm start

Go to web browser and you can run from http://127.0.0.1:3000
